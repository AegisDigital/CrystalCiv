A civilization-building strategy game with fantasy aspects in Discord.

## Run
> See [Sentry](https://github.com/samueleaton/sentry)

With Sentry:
```bash
./sentry
```

Or (Without Sentry):
```bash
crystal src/CrystalCiv.cr
```
